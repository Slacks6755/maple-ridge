# Maple Ridge Construction & Development

Website for Maple Ridge Construction in Etowah, TN — licensed general contractor serving McMinn, Bradley, Polk, and Monroe Counties in Southeast Tennessee.

**Live site:** https://www.mapleridgedev.com
**Owner:** Zack Newman · (423) 506-2380 · zack@mapleridgedev.com

## Stack

Astro 5 (static) · Tailwind CSS 3 · deployed on Netlify from `main` (build → `dist/`).

```sh
npm install
npm run dev       # local dev server
npm run build     # production build to dist/
npm run preview   # preview the build
```

## Pages

Home · About · Contact (Netlify Forms) · Projects (single-page portfolio with before/after) · 3 service pages (New Construction, Remodeling, Land Development) · Service Areas index + 5 city pages (Etowah, Athens, Cleveland, Madisonville, Sweetwater) · 404 · generated `sitemap.xml`.
